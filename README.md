# Getaway — Vacation Planner 2026

A premium vacation planning tool for late March/early April 2026 — combining ski resort live conditions with warm-weather destination research.

## Features

- **Dual-Mode Planning**: 8 ski resorts with live snow conditions + 13 warm-weather destinations
- **Interactive Map**: Leaflet-powered map with toggleable snow/warm markers
- **Side-by-Side Comparison**: Compare up to 3 destinations across temperature, flights, hotels, and more
- **Budget Estimator**: Calculate trip costs for any destination
- **Shortlist & Planner**: Bookmark favorites and plan your trip
- **Dark Mode**: Full light/dark mode support
- **Mobile Responsive**: Works on all screen sizes

## Tech Stack

- Vanilla HTML/CSS/JS (no build step)
- [Leaflet](https://leafletjs.com/) for maps
- [Chart.js](https://www.chartjs.org/) for comparison charts
- [Lucide Icons](https://lucide.dev/)
- [Fontshare](https://www.fontshare.com/) fonts (Satoshi + Zodiak)

## Data Sources

- Ski conditions fetched via Firecrawl from official resort pages (March 13, 2026)
- Warm destination research aggregated from travel sources and Google Maps
- Flight estimates based on Delta/JetBlue/American from JFK
